About PuzzleVerse AI

Welcome to PuzzleVerse AI, a luxury, offline-capable puzzle canvas designed to turn any image into an engaging, interactive game layout.

Key Capabilities

  • Procedural Audio: Audio synthesizers create real-time chimes, clicks, block slides, and relaxing looping background pads via the Web Audio API.
  • Geometric Jigsaw Masks: High-fidelity mathematical Bezier equations carve out authentic tabs and slots.
  • Upload Studio: Pan, crop, scale, and compress custom image uploads on a client-side canvas.
  • 5 Puzzle Modes: Supports Classic Jigsaw, Rotational puzzle, Sliding Blocks, Card Memory matching, and Speed countdown races.
  • Persistent Savings: Scores, best times, settings configurations, and custom milestones save locally inside LocalStorage.

Built with 100% Vanilla JavaScript (ES6+), CSS3, and HTML5 Canvas API. Designed for performance, accessibility, and fluid responsiveness.
